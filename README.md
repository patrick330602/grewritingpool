# Helper for GRE Writing Pool

Because why not?

## Install

`pip install grewritingpool`

or install from source:
`python3 setup.py install`

## Usage

As a API:
```python3
from grewritingpool import fetch
json = fetch.fetch_type("issue")
```

As a executable:
```bash
grewriting [all|issue|argument]
```
